# 🌊 StudyWave 

**StudyWave** is a full-stack, gamified learning platform designed to make education engaging, structured, and collaborative. Built with the MERN stack, it features real-time chat, an AI-powered doubt assistant, interactive quizzes, and a gamified task management system to keep students motivated.

![Landing Page](./assets/Screenshot%202026-06-30%20170915.png)
**Live Demo:** [Click here to view the app](https://tracker-ksb7.vercel.app/)
---

## ✨ Key Features & Showcase

### 🔐 Secure Authentication & Role Management
Role-based access separating Student and Admin portals. Admins can create tasks, manage quizzes, and upload resources, while students focus on learning and tracking their progress.

![Login Page](./assets/Screenshot%202026-06-30%20170926.png)

### 🎮 Gamified Task Management
A dynamic dashboard where students can view pending tasks, complete them before deadlines, and earn points. Earning points pushes students higher up the global leaderboard.

![Dashboard & Tasks](./assets/Screenshot%202026-06-30%20171008.png)

### ❓ AI Mentor (Gemini API)
Stuck on a concept? The integrated AI Mentor uses Google's Gemini API to provide step-by-step, personalized guidance for the toughest questions without leaving the app.

![AI Mentor](./assets/Screenshot%202026-06-30%20171101.png)

### 📅 Personal Timetable
An interactive calendar system allowing students to plan their study schedules, add quick tasks, and manage their daily learning goals.

![Timetable](./assets/Screenshot%202026-06-30%20171049.png)

### 🧠 Interactive Quizzes
Admins can deploy time-sensitive quizzes. Students take them in real-time, earning immediate feedback and points that contribute to their overall rank.

![Quizzes](./assets/Screenshot%202026-06-30%20171026.png)

### 💬 Real-Time Chat
Built with Socket.io, the integrated messaging system allows students to instantly connect, discuss coursework, and collaborate with their peers.

![Chat](./assets/Screenshot%202026-06-30%20171206.png)

## 🛠️ Tech Stack

**Frontend:**
* React.js (Vite)
* Tailwind CSS (Styling)
* Framer Motion (Animations)
* Axios & React Router

**Backend:**
* Node.js & Express.js
* MongoDB Atlas & Mongoose (Database)
* JSON Web Tokens (JWT) & bcryptjs (Authentication)

**Real-Time & AI:**
* Socket.io (Real-time bi-directional communication)
* Google Generative AI / Gemini API (Doubt resolution)

**Deployment:**
* Vercel (Frontend)
* Render (Backend REST API & Socket Server)
